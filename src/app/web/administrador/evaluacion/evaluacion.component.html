Slacker Style Modal

Every go to Slacker Radio and think that the modal they use is kidna nice? Wanted to mimic this feature? Now you can.

I tried to make this as easy as possible by making it so you only have to add a class, similar to how bootstap added modal-lg now there is modal-slacker. This means you still have access to bootstraps orginal modals!

Looks best in fullscreen mode. To view this snippet in fullscreen click on the button in the top right hand corner or if you are in fullscreen click on the same button to return to Bootsnipp!
